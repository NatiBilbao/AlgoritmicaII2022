# ALGORÍTMICA II
El presente repositorio tiene la finalidad de ser una herramienta para almacenar y brindar acceso a los distintos algoritmos, estructuras de datos y ejemplos basicos; aprendidos en el transcurso del todos los términos de la materia de Algorítmica II.

Cada algorítmo y/o estructura de datos contiene un archivo **README** separado donde se da una breve explicación del funcionamiento de dicho algorítmo; se indica además el código base del mismo en distintos lenguajes de programación, algunos ejemplos de aplicación, incluyendo también enlaces a páginas web y videos que desarrollan más los conceptos del algoritmo mostrado.
